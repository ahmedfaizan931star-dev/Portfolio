ScanMate AI Pro

Document scanning with OCR & AI workspace

Coming soon on Play Store

Problem

Paper documents need to be digitized, searchable, and sometimes protected — but most scanner apps stop at "take a photo" and leave organization, security, and text extraction to the user.

Solution

Built a full document management system on CameraX and ML Kit: real-time edge detection with auto-deskew, on-device OCR with paragraph reconstruction, full-text search across every scan, AES-128 password-protected PDF export, QR/barcode generation, and a biometric-locked vault for sensitive documents — all on a reactive Room/Flow architecture that works fully offline. A Gemini-powered AI workspace layers on top for summarizing, translating, and chatting with scanned document text.

Result

Taken through 15 rounds of iterative production hardening, from a 6.4/10 initial review to a 10/10 production-ready build — now also handling business card scanning with direct vCard extraction and automated document quality scoring on capture.

  • CameraX
  • ML Kit
  • Room
  • Flow
  • iTextG
  • Biometrics
  • Gemini API

SpendWise

AI-powered expense & subscription tracker

Coming soon on Play Store

Problem

Recurring subscriptions and everyday spending quietly add up, and manually categorizing every purchase or typing in receipt details is tedious enough that most people just stop tracking.

Solution

Built an offline-first tracker where the Gemini API handles smart expense categorization and ML Kit OCR (carried over from ScanMate) reads receipts automatically, pulling amounts and merchant details without manual entry. All data stays local via Room — no cloud dependency.

SG

Signal

Single-file Gemini-powered chat app

Problem

Most AI chat interfaces require a framework, a build step, and hosting infrastructure — overkill for a lightweight, personal Gemini API client that just needs to work anywhere.

Solution

Built Signal as a single self-contained HTML/CSS/JS file — no framework, no build step. Added markdown rendering via marked.js, syntax highlighting via highlight.js, sanitized output through DOMPurify, Google Search grounding, and temperature controls.

Result

A fully-featured chat client with per-message edit, regenerate, delete and copy actions, plus chat pinning, sidebar search, and rename — deployable by dropping one file anywhere.

  • Gemini API
